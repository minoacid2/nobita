နှင်းဆီလှိုင်

#အောင်မှူးဝေ
{ ၁ }
စန္ဒရားခလုတ်တွေပေါ်
ပြင်သစ်ရေမွှေးနံ့နဲ့
ပြင်သစ်ကဗျာတစ်ပုဒ်ဟာ
မြန်မာလိုအသံထွက်ပြီး ပြေးနေ။
`ကျွန်တော် ဘာလုပ်ပေးရမလဲမမလှိုင်´
မျက်စောင်းက
၈၄လမ်းပေါ်မှောက်သွားတဲ့ သုံးဘီးကားလို
ကျွန်တော့်နှလုံးသား လမ်းဘေးတမာပင်လေးကိုဝင်တိုက်တယ်။

{ ၂ }

ကျွန်တော်က
လက်ဖက်ရည်တစ်ခွက်တန်ကဗျာသရဲပါမမလှိုင်
``ရို့စ်အယ်လ်ရေ နန်းရှေ့ ပီဇာဆိုင်မှာ´´
ဆိုတဲ့
ဂျင်မီရဲမြင့်တို့လို
ကျောက်စိမ်းလည်ဆွဲကြီးလည်းမပေးနိုင်
ကျောက်ဝိုင်း၊ရဟိုင်းမှာ
အိပ်မက်အသေကြီးထမ်းပြီး
လမ်းမီးတိုင်အောက် တောက်ခေါက်နေရတဲ့ကောင်စားပါ။

{ ၃ }

`ပိုက်ဆံအစုတ်တွေ
စာအုပ်အဟောင်းတွေ
အိမ်ထဲယူယူမလာနဲ့အောင်အောင်
ကျက်သရေမရှိဘူး´
ဟုတ်ကဲ့
ကျွန်တော် ဗဟိုဘဏ်ကိုမုန်းမိပါတယ်မမလှိုင်
စာအုပ်ဟောင်းလေးတွေကို ခွင့်လွှတ်ပါ
သူတို့ဟာ
နိုင်ငံတော်ဦးကင်းက ဥာဏ်ပညာသရဖူအတွက်
အဖမ်းခံရသူ ခေတ်သုံ့ပန်းတွေပါ။

{ ၄ }

ဒဏ်ရာတွေက လင်းလင်းချင်းချင်းပဲ မမလှိုင်
ကြယ်တွေကိုမော့ကြည့်ရင်း
မြောင်းထဲပြုတ်ကျတဲ့ လူကုံထံရပ်ကွက်ထဲ
ဟိုချီမင်းကဗျာတွေ ဖတ်ခွင့်မရခဲ့သူ
၁၆နှစ်အရွယ်ကောင်ကလေး
မမလှိုင်မာမီနဲ့စျေးမလိုက်ချင်ဘူး
ရွှေလက်ကောက်တစ်တောင်လောက်နဲ့
ခရမ်းချဉ်သီးအူပေါက်တွေ အဆစ်တောင်းတောင်းနေတာ
ကျွန်တော်ရှက်တယ်။

{ ၅ }

ကျွန်တော်တို့လူငယ်ဘဝက အခုပွင့် အခုကြွေ
လေပြည်လေပြင်းများစွာရဲ့သားကောင် ခရေ၊
စာအုပ်တစ်အုပ်ဖတ်တာကအစ
လူထုခေါင်းဆောင်ပုံကလေးဝယ်တာအဆုံး
အားလုံးစွန့်စားခဲ့ရတယ်၊
မျှားစာအသိုးတွေနဲ့
ငါးမျှားချိတ်တွေချည်းချက်စားခဲ့ရတဲ့
ညစာစားပွဲတွေချည်းပါ မမလှိုင်။
အဲ့ဒီနှစ်တွေက
သောင်းကျော်တန်တဲ့နှုတ်ခမ်းဆိုးဆေးကလေးဆိုးပြီး
`မသိဘူး မသိဘူး´လို့ပြောခဲ့တာ
နွေပေါင်းများစွာ မေ့မရဘူး
ပူတယ်။

{၆}

ပါတော်မူနေ့ဆို တစ်အိမ်လုံးမီးခိုးတိတ်
မြတောင်တိုက် သွားစားတဲ့လူချမ်းသာကြီးတွေ
အိမ်ဖော်နဲ့ကျွန်တော်လိုကပ်ပါးကောင်က ထမင်းငတ်ပြီး
အယ်လ်ဇေးရှင်းခွေးကြီးတွေတော့
အမဲသားနဲ့ပေါ်ဆန်းမွှေးထမင်းနဲ့
ခွေးစာထမင်းထဲစပါးလုံးပါရင်
ဖော့တောရွာသူအိမ်ဖော်မလေး
သေလုအောင် အရိုက်ခံရတယ်
လူဆိုတာခွေးထက်မြတ်တယ်လို့
သက်သေမပြနိုင်ခဲ့တဲ့နှစ်များပါပဲမမလှိုင်။

{၇}

ပိုက်ဆံကြောင့် ကမ္ဘာကြီးကရှင်သန်နေတာလို့မမလှိုင်ပြောတယ်
သေသွားတဲ့သူငယ်ချင်းတွေရဲ့အသက်ကို
ပိုက်ဆံနဲ့ကျွန်တော်မဝယ်နိုင်ခဲ့ဘူး။
မမလှိုင်ညီမလေး
ဒရိုင်ဘာ ရဲဘော်လေးနောက်လိုက်ပြေးတဲ့သတင်းလည်း
ပိုက်ဆံကန့်လန့်ကာကြီးနဲ့
လုံအောင်မကွယ်နိုင်ခဲ့ဘူး။
ပိုက်ဆံနဲ့ဂုဏ်သိက္ခာကိုဝယ်မရလို့
လူမသိသူမသိသတ်သေသွားတဲ့သူဌေးတွေ
နှင်းဆီလှိုင်တို့အသိုင်းအဝိုင်းမှာ ပုံလို့
ကြွေးကြော်ရဲပါသလား မမလှိုင်။

{ ၈ }

အချစ်ဆိုတာ မာဇဒါဂျစ်အစိမ်းလေးပေါ်မှာ သဇင်
မြေပြင်ပေါ်မှာတော့
စားပြီးလွှင့်ပစ်လိုက်တဲ့ချဉ်ပေါင်းအိတ်ခွံလေးပဲ မမလှိုင်၊
လူဆင်းရဲတွေအတွက်
ညမနက်စာအဆင်ပြေမှ
အချစ်ရထားက ဥသြဆွဲတာမျိုး။
မမလှိုင်အဖေက သိပ်ညောင်းတတ်တာပဲ
သူအညောင်းပြေတိုင်း
အိမ်ဖော်မလေးတွေ ဗိုက်တစ်လုံးနဲ့တောပြန်ပြေးခဲ့ရ၊
`ပိုက်ဆံခိုးလို့နှင်ထုတ်လိုက်တယ်´
`လက်ဆော့လို့လေ´
လူကုံထံတွေကတော့ တစ်ကိုယ်လုံးဆော့လည်း
အပြစ်မရှိကြပါဘူး။
ကိုယ့်လက်မှတ်တောင် ကိုယ်မထိုးတတ်တဲ့
မမလှိုင်အဖေကြီး
ဘယ်လိုချမ်းသာလာသလဲဆိုတာ
ပုဏ္ဏားကုန်းက အကြားအမြင်ဆရာတောင်မသိပါဘူးလေ။

{၉}

ဂီတက
တခြားသူတွေဆီ ပျော်ရွှင်မှုကိုသယ်ဆောင်ပေးတယ်
ဆိုပေမဲ့
ရတနာခုနစ်ပါးနဲ့မော်တော်ကားလည်းစီးတယ်ဆိုတာကျ
မပျော်ရပါလား
အရှေ့တောင်အာရှမှာ အချမ်းသာဆုံးနိုင်ငံက
သူတောင်းစားလက်မှတ်ကလေးကိုင်ပြီး
နိုင်ငံတကာက ငွေပိုလေးတွေ လျှောက်တောင်းရတဲ့နှစ်များမှာ
ကျွန်တော်လူလားမြောက်ခဲ့ပါတယ် မမလှိုင်
ကတ္တီပါတော်လှန်ရေး
မြစိမ်းပြာတော်လှန်ရေး
ရွှေဝါရောင်တော်လှန်ရေး
စိတ်ဓာတ်က
ရေဆိုးမြောင်းထဲဆွေးနေတဲ့လူတွေထဲ
သစ္စာတရားဟာ
ခေတ်အဆက်ဆက် လင်ကောင်မပေါ်တဲ့မိန်းမတစ်ယောက်လို
ဖာတန်းပဲသွားရမလား တမလွန်ဘုံပဲသွားရမလား
ဒီလိုနဲ့
ကျွန်တော်တို့တော်လှန်ရေးတွေ မအောင်မြင်ခဲ့ဘူး
သပိတ်မှောက်ရင်မှောက်
မဟုတ်ရင်
ဗိုက်မှောက်နေရသူတွေရဲ့ နိုင်ငံမှာပေါ့ မမလှိုင်။

{၁၀}

နှစ်ငါးဆယ်ဆိုတာ ဘာကြာလို့လဲ
အသံက သစ်ဆွေးတုံးထဲမှာခုန်နေတယ်။
တစ်တောလုံးလည်း တောခြောက်သံနဲ့
စမ်းရေထဲ သွေးပျစ်ပျစ်နှစ်နှစ်တွေမျောနေတာနဲ့
မျက်နှာငယ်ရခြင်းအကြောင်းကို စာစီကုံးပါ နှစ်တွေ။
``ကျွန်တော်တို့နိုင်ငံသည်
လှောင်ပြောင်ရယ်မောသံများ၏အလယ်ဗဟိုတွင်တည်ရှိပါသည်…´´ဆိုတာကို
ငိုသံပါကြီးနဲ့အော်ကျက်ခဲ့ရတဲ့နှစ်တွေ
စာမေးပွဲတော့ အောင်ခဲ့တာပေါ့ မမလှိုင်။

{၁၁}

ငရဲပန်းက ပိုလှ ပိုမွှေးတော့
ရွေးချယ်စရာ နည်းပါးလွန်းသူတွေမှာ
နေကာမျက်မှန်တကားကားနဲ့
ခေတ်ကျိုးခေတ်ပဲ့တစ်လျှောက် ကားရားကားရား ပန်ဆင်သွားကြတယ်
လက်တွဲဖော်ကို
စျေးချိုတော်မှာအထည်ရွေးသလို
မမလှိုင်ရွေးသွားတာ မွှေးပါတယ်
သတင်းတွေပေါ့။
အခါတော်ပေးတဲ့နေ့က
မိုးရွာတယ် ။
